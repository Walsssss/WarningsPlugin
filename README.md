# WarningsPlugin
Author: Walson

## COMMANDS
/warn <player> <reason>       - Warn a player 
/warns <player>               - List a player's warns (text)
/removewarn <player> <n|all>  - Remove a warning
/walsonwarn panel             - Open warnings panel GUI
/walsonwarn reload            - Reload config and messages
/walsonwarn help              - Show this help

Permissions can be found on [here](https://github.com/Walsssss/WarningsPlugin/blob/main/src/main/resources/plugin.yml)
